How do I import the Pandas library and load a dataset in Google Colab?
import pandas as pd
df = pd.read_csv("filename.csv")
How do I print the first 5 rows (default) and the first N rows (e.g., 10)?
df.head()    # Print first 5 rows (default)
df.head(10)  # Print first 10 rows
How do I print the last 5 rows (default) and the last N rows (e.g., 10)?
df.tail()    # Print last 5 rows (default)
df.tail(10)  # Print last 10 rows
How do I print the datatype of each column?
print(df.dtypes)
How do I find the total number of columns?
print("Total number of columns:", len(df.columns))
How do I count missing values in each column?
df.isnull().sum()
How do I find the total number of rows?
print("Total number of rows:", len(df))
How do I print columns that have the 'object' datatype?
print(df.dtypes[df.dtypes == 'object'])
How do I find the memory usage of the DataFrame?
print(df.memory_usage())
How do I add headers to a CSV file that lacks column names?
Steps: Create a list of header names, then use the *names* parameter in `pd.read_csv()`.
import pandas as pd

# Step 1: Give names to columns
headers = ["id", "company", "fuel", "doors", "price"]

# Step 2: Add these headers while reading the file
df = pd.read_csv("cars.csv", names=headers)

# Step 3: Show data
print(df)
How do I replace missing value symbols (like "?") with NaN AFTER loading the dataset?
import numpy as np
df.replace("?", pd.NA, inplace=True) 
# or 
df.replace("?", np.nan, inplace=True)
How do I identify missing values using `isnull()` and `notnull()`?
`isnull()` returns True for missing, False otherwise. `notnull()` does the opposite.
# Check missing values for the whole DataFrame
df.isnull()

# Check missing values column-wise
df.isnull().sum()

# Identify present values
df.notnull()
How do I find rows that contain missing values, or rows that do NOT contain missing values?
# Rows that contain ANY NaN value
df[df.isnull().any(axis=1)]

# Rows that do NOT contain ANY NaN value
df[df.notnull().all(axis=1)]
How do I delete a single row or multiple rows by index?
df = df.drop(index=1)      # Delete row with index 1
df = df.drop(index=[0, 2]) # Delete multiple rows
How do I delete rows based on a specific condition?
df = df[df["price"] >= 13000] # Keep only rows where price >= 13000
How do I delete rows that have ANY NaN value, or where a specific column has NaN?
df = df.dropna()               # Delete rows that have ANY NaN value
df = df.dropna(subset=["price"]) # Delete rows where price column has NaN
How do I split a dataset into training and testing sets (70% train, 30% test)?
from sklearn.model_selection import train_test_split

# Choose X (features) and y (target column)
X = df.drop("price", axis=1)
y = df["price"]

# Split into 70% train and 30% test
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)
How do I train a model and print Accuracy, Precision, Recall, F1-score, and Support?
from sklearn.metrics import classification_report
from sklearn.tree import DecisionTreeClassifier

# Assume X_train, X_test, y_train, y_test are ready

# Step 1: Train model
model = DecisionTreeClassifier()
model.fit(X_train, y_train)

# Step 2: Predictions
y_pred = model.predict(X_test)

# Step 3: Print Metrics
print(classification_report(y_test, y_pred))
